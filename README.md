# redpack

⭐️ TG群红包扫雷机器人

😌 介绍：
复刻经典扫雷游戏，支持双语，中文和英语。

😁 特点：
✅ 带红包是否中雷预警\n
✅ 带可控制用户是否有雷\n
✅ Python语言编写运行高效配置简单\n
✅ 带假人自动抢包\n
✅ 带邀请返利 自动抽水\n
✅ 附有完整管理命令\n

😋 更多功能可定制，演示群 https://t.me/iluoboya
